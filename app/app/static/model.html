Model: Yolov5x

We used Yolov5 model using data from the Kaggle SIIM-FISABIO-RSNA COVID-19 Detection competition.

Model Name: exp240-best.pt
Status: Stable Training

Purpose

This main purpose of this app is to detect and localize COVID-19 on chest x-rays. It classifies each image into one of two classes:
- Negative for pneumonia
- Typical for COVID-19

Input

JPG or PNG images (No DICOM or TIFF, unsupported).

Output

Two predicted classes - Negative for pneumonia or Typical for COVID-19. When the app predicts 'Typical for COVID-19', it also draws bounding boxes around areas where it detected opacity.

Dataset Summary

The original dataset has four classes:
- negative for pneumonia
- typical, atypical and indeterminate for COVID-19
